## Install

Install with npm:

```bash
npm install --save-dev packjs
```

Install with yarn:

```bash
yarn add packjs --dev
```

packjs is an intelligent bundler for modules. It can generate complex webpack configuration according to simple configuration, and automatically install the required dependencies, support HTTPS certificatesã€‚
